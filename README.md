# Claude Code Plugins

A collection of plugins for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Installation

Add this marketplace to Claude Code:

```
/plugin marketplace add grahamcarlyle/claude-code-plugins
```

Then install individual plugins:

```
/plugin install <plugin-name>@grahamcarlyle-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [project-plan](project-plan/) | Sync Claude plan-mode plans with markdown files in your project |
